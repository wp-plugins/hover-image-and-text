== Description ==

Hover Image makes you able to add hover images/text and read more button to your WordPress Posts and Pages. Use the [hover] sort code to display the images uploaded from admin.

== Installation ==

1. Upload `hover` folder to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Place [hover]  in your post/page
